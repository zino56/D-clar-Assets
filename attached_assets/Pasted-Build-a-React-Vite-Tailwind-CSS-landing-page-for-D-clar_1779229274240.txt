## TECHNICAL DETAILS:

- Framework: React 18 + TypeScript + Vite
- Styling: Tailwind CSS 3.4
- Packages: `shaders` (Swirl, ChromaFlow, FilmGrain, FlutedGlass from `shaders/react`), 
  `lucide-react` (ArrowRight, Clock, Menu, X, FileUp, Cpu, CheckCircle2, FileText, 
   Brain, Calculator, Shield, ChevronRight)
- Font: General Sans via Fontshare (loaded in index.html)
- Live clock: updates every second, shows Africa/Algiers timezone in HH:MM format
- All hover animations: duration-500, ease-[cubic-bezier(0.25,0.1,0.25,1)]
- Text-roll buttons: overflow-hidden h-[20px] flex-col container, 
  group-hover:-translate-y-1/2, two text spans stacked vertically
- Max content width: 1440px centered with mx-auto
- Responsive breakpoints: default Tailwind (sm:640 md:768 lg:1024 xl:1280)
- Arabic text sections: dir="rtl", text-right, font General Sans (supports Arabic)
- No chat interface, no free-form inputs — buttons only
- Arabic numerals in Arabic sections (use Eastern Arabic: ٢٠٢٥/٠٤٧ for document refs)
