Build a React + Vite + TypeScript + Tailwind CSS authenticated SaaS dashboard for "Déclaration Copilot" — an AI-powered declaration-readiness workspace for Algerian accountants.

This is NOT a marketing page. This is the logged-in product experience after the user clicks "Démarrer" / "Commencer".

The app should feel like:
- a premium AI workspace
- a serious accountant operating system
- inspired by Hermes Workspace desktop layout
- dense but elegant
- built for action, review, anomaly fixing, and declaration readiness

Use:
- `lucide-react` for icons
- `framer-motion` for panel transitions, section transitions, number changes, and agent panel animations
- Tailwind CSS only
- React Router for internal routes
- NO backend integration yet; use realistic mock data in TypeScript objects
- Layout must be desktop-first but responsive down to tablet and mobile
- All screens must be production-quality UI, not wireframes

-----------------------------------
AI AGENT CONCEPT
-----------------------------------

Do NOT make this a generic chatbot panel.

Instead, model it as a multi-agent workspace with specialized skills.

Each agent card must show:
- agent name
- role
- status
- supported skills
- allowed outputs

Use these agent definitions:

1. **Classifier Agent**
   - role: identifies document type
   - skills: invoice classification, declaration relevance, confidence scoring
   - outputs: invoice / delivery note / receipt / unknown

2. **Extractor Agent**
   - role: pulls structured data from OCR text
   - skills: NIF, RC, Article d’imposition, date, supplier, amounts, payment method
   - outputs: structured JSON field set

3. **Fiscal Checker Agent**
   - role: verifies readiness logic
   - skills: missing field review, anomaly explanation, TVA consistency review, legal-risk surfacing
   - outputs: warnings, blocking issues, confidence notes

4. **Follow-up Writer Agent**
   - role: drafts supplier/client follow-ups
   - skills: French drafting, Arabic drafting, critical tone adjustment, urgency wording
   - outputs: FR message, AR message, shorter reminder

5. **Export Agent**
   - role: prepares declaration-ready export bundles
   - skills: period grouping, READY filtering, accountant summary drafting
   - outputs: export preview, grouped rows, readiness summary

6. **General Copilot**
   - role: orchestration and explanation assistant
   - skills: explain anomalies, suggest next actions, guide accountant through workflow
   - outputs: guidance cards, next steps, review checklist

For each agent, provide a “skills” mini-panel in the right workspace with tags and a recent action feed.
